
# 【MHWilds】クエスト・チャレンジ・ジェネレータ

## Overview

![](https://img.shields.io/badge/Hosting-Github_Pages-blue)

ページを開くたびに【モンスターハンターワイルズ】のランダムなクエスト・チャレンジ画像を生成してくれるウェブサービスです。
マルチで遊んでいる時に、なかなか次のクエストが決まらない。そんな時に使ってみてください。

生成された画像は右クリックでコピーできるように作りましたので、そのまま Discord のチャット欄に貼りつけることができます。
実際に生成される画像はこんな感じです。

![](doc/sample.png)

チャレンジ機能は話のタネになればと思い付けた機能ですので、難しかったり達成不可能ならば、そのまま無視していただいても構いません。
